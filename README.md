## 🤘 Sobre

Repositório do projeto de testes automatizados do sistema Startbugs, construído no curso BDD eXperience!

## 💻 Tecnologias
- Ruby
- Capybara
- RSpec
- Cucumber
- Selenium WebDriver

## 🤖 Como executar

1. Clonar o repositório, instalar as dependências
```
bundle install
```

2. Executar testes de regressão
```
bundle exec cucumber
```
